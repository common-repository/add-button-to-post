== Description ==

"Add Button To Post" is a plugin providing social button. Hatena, Twitter, Facebook, Google. Especially, this plugin is optimized for Japanese.

== Installation ==

1. Upload the `child-themify` directory and its contents to the `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
